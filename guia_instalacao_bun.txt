Guia de Instalação do Bun
================================

O que é o Bun?
--------------
O Bun é um runtime JavaScript moderno, semelhante ao Node.js, mas mais rápido e com suporte nativo a TypeScript, pacotes npm e APIs da Web.

Instalando o Bun
----------------

1. Pré-requisitos:
   - Ter acesso ao terminal (Linux, macOS ou Windows com WSL ou Git Bash).
   - Ter o `curl` ou `powershell` instalado.

2. Instalação no Linux/macOS:
   Abra o terminal e rode o comando:
   
   curl -fsSL https://bun.sh/install | bash

   Após a instalação, feche e abra novamente o terminal.

3. Instalação no Windows (PowerShell):
   Abra o PowerShell e rode o comando:
   
   powershell -c "irm bun.sh/install.ps1 | iex"

4. Verificar se deu certo:
   Rode no terminal (ou reinicie o powershell): 
   
   bun --version

   Se aparecer um número de versão, o Bun foi instalado corretamente.

Links úteis:
------------
- Site oficial: https://bun.sh
- Documentação: https://bun.sh/docs

Pronto! Agora você já tem o Bun instalado no seu computador.
